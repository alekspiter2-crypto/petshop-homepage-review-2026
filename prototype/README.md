# Petshop.ru — облегчённая главная

Адаптивный кликабельный прототип главной страницы Petshop.ru для десктопа и iPhone 17 Pro. Вторая концепция построена по логике Ozon, но в фирменной стилистике Petshop.

## Как открыть

Откройте `index.html` в браузере. Внешние библиотеки не нужны: стили, сценарии и изображения находятся внутри папки прототипа.

## Контрольные размеры

- Десктоп: 1440 px по ширине.
- iPhone 17 Pro: 402 × 874 CSS px (1206 × 2622 физических пикселей при масштабе 3×).

## Принцип новой версии

- поиск — главный элемент шапки;
- один компактный промобаннер вместо большого набора конкурирующих блоков;
- быстрые сценарии собраны в одну горизонталь;
- товары начинаются уже на первом экране и продолжаются бесконечной лентой;
- между товарными рядами встроены компактные рекламные полоски;
- изображения в товарных карточках имеют пропорцию 3:4;
- карточки поддерживают просмотр трёх кадров по наведению или касанию;
- рейтинг и количество отзывов видны на десктопе и мобильном;
- персонализация сохранена, но не занимает отдельную панель;
- на мобильном используется отдельная компактная композиция.
- используется официальный векторный логотип Petshop.ru;
- высота мобильного промобаннера уменьшена до 184 px.

## Что работает

- десктопное и мобильное меню;
- поиск с обратной связью;
- добавление товаров в корзину и избранное;
- повтор прошлого заказа;
- мобильная нижняя навигация;
- автоматическая подгрузка новых товарных полок при прокрутке;
- адаптивная сетка без горизонтального переполнения страницы.
